# Фронтенд СПК «Хорошово-1» с кабинетами по ролям

Этот архив содержит только часть `frontend/` портала — с продуманным дизайном и системой личных кабинетов по ролям.

## Установка

1. Распакуйте архив в папку `frontend` вашего проекта **или** используйте как отдельный проект:

```bash
npm install
```

2. Создайте файл `.env`:

```env
VITE_API_URL=http://localhost:8000
```

(или укажите URL вашего backend на Render / Timeweb).

3. Запуск:

```bash
npm run dev
```

Сайт откроется на `http://localhost:5173`.

## Роли и кабинеты

На backend поле `role` пользователя может принимать значения:

- `admin` — администратор портала
- `chairman` — председатель
- `accountant` — бухгалтер
- `auditor` — ревизионная комиссия
- `board` — член правления
- `member` — обычный садовод

В зависимости от роли меняются:

- название и содержимое личного кабинета (`/cabinet`);
- подпись в карточке пользователя в сайдбаре.

## Основные разделы

- Главная — новости и короткая сводка
- Новости — все официальные объявления
- Документы — устав, протоколы, решения (список из API)
- Взносы — раздел «Мои взносы» (после входа)
- Голосования — список опросов
- Карта участков — упрощенный список
- Форум — темы обсуждений
- Личные кабинеты по ролям — страница `/cabinet`
